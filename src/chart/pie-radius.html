This sample compares countries by population density and total area using various radius in a pie series.

In this example, you can see how to render donut chart with different radius. You can use radius mapping property to achieve this feature. dataLabel is used to represent individual data and its value.


Tooltip is enabled in this example. To see the tooltip in action, hover a point or tap a point in touch enabled devices.


More information on the various radius in pie series can be found in this documentation section.
