This sample shows the gold medal count scored by each country at the summer olympics 2024 games using smart labels on the chart.

In this example, you can see how to arrange the labels smartly without overlapping with each other. You can use enableSmartLabels property to enable or disable the support.

Tooltips are enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.


More information on the smart label in pie series can be found in this documentation section.
